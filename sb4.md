**C语言的每一句结尾都要有**`;`**!**

<!--more-->

使用scanf函数接收外部输入时，传入的参数前要加`&`

```c
scanf("%d %d",&a,&b);
```

c语言中的整数相互运算结果还是整数，会向前取整，所以需要有小数运算的地方要定义为浮点型
c的浮点型为`float`,`double`，格式化符使用`%lf`

```c
printf("%lf",10.0);
```

c的运算符优先级，单目最高

```c
//单目运算
c*-b //c乘以-b
c*+b //c乘以+b
```
