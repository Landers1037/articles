# 首次安装完建议先更新源

```bash
sudo apt-get update
```

<!--more-->备份源列表

```bash
cp /etc/apt/source.list /etc/apt/source.list.bak 
```

查看系统版本

```bash
lsb_release -a
```

我是16.04位xenial

换源

```bash
sudo nano /etc/apt/source.list
#删除所有内容
```
